# {frontmatter.title}

一个用于 Astro 项目的美观实用的组件库，为同一个组织下的多个项目提供统一的 UI 风格。

## 特性

- 🚀 基于 Astro 构建，性能优异
- 📦 组件丰富，满足常见需求
- 📱 响应式设计，完美适配各种设备
- 🔧 TypeScript 支持，开发体验优秀

<InstallTabs packageName="@coffic/cosy-ui" />
